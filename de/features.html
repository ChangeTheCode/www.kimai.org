Was kann Kimai für Sie tun?

Kimai bietet noch viel mehr als nur die Zeiterfassung, hier finden Sie eine Liste weiterer wichtiger Business-Funktionen.

Mobil einsetzbar

Das Layout ist vollständig responsiv und für den Desktop und mobilen Einsatz konzipiert. Jeder Benutzer kann die anzuzeigenden Datenspalten konfigurieren, z. B. um horizontales Scrollen auf kleinen Geräten zu verhindern.

Plugins

Kimai wurde mit Fokus auf Erweiterbarkeit entwickelt, um Funktionen per Plugins hinzufügen zu können. Entwickler, die eigene Plugins für Kimai erstellen möchten, können auf eine umfangreiche API zugreifen.

Mehrere Benutzer

Kimai kann von beliebig vielen Benutzern verwendet werden: Jede Installation unterstützt eine unbegrenzte Anzahl von Benutzern mit unterschiedlichen Rollen (siehe Berechtigungen).

Sprachen

Kimai kann in jede Sprache lokalisiert werden und ist bereits in mehr als 30 Sprachen übersetzt.

Teams

Mit Teams teilen sie Ihre Benutzer in logische Gruppen ein, denen Kunden und Projekte zugewiesen werden können, was die Sichtbarkeit und den Zugang für andere Benutzer einschränkt.

Kunden

Verwalten Sie Ihre Kunden und legen Sie Adresse, Kontakte, Land, Währung, Sichtbarkeit (zur Archivierung) und viele weitere Felder fest.

Projekte

Verwalten Sie Projekte für jeden Kunden mit Auftragsnummer, Budget und Sichtbarkeit (zur Archivierung).

Tätigkeiten

Tätigkeiten werden Zeiteinträgen zugewiesen und können zur Auswertung und Filterung verwendet werden. Tätigkeiten können global sein, keine Duplizierung notwendig, wenn mehrere Projekte dieselbe Tätigkeit nutzen.

Rechnungen

Verschiedene Rechnungsvorlagen, Möglichkeiten zur Gruppierung von Einträgen, konfigurierbare Rechnungsnummern und Sie können eigene Vorlagen in den Dateiformaten PDF, DOCX, HTML, XLSX und ODS hinzufügen.

Daten Export

Stundenzettel können zur Verwendung in anderen Programmen oder zur gemeinsamen Nutzung in verschiedenen Formaten exportiert werden. Exportierte Datensätze werden gesperrt und vor weiteren Änderungen geschützt.

Ausgaben

Behalten Sie den Überblick über Ihre Ausgaben basierend auf Kunde, Projekt und Aktivität. Diese Ausgaben können kategorisiert und in Ihre Rechnungen aufgenommen werden. Die Spesen- und Ausgabenverwaltung ist als kostenpflichtiges Plugin verfügbar.

SAML

Die können Benutzer über SAML (Single Sign-On, auch bekannt als SSO) anmelden bzw. registrieren und dafür Ihren vorhandenen Identitätsanbieter wie Google Workspace, Authentik oder Azure Active Directory nutzen.

JSON API

Mit einer umfangreichen JSON-API zum Lesen und Schreiben von Daten, so dass externe Anwendungen und Ihre benutzerdefinierten Tools mit Ihrem bevorzugten Time-Tracker kommunizieren können.

Zwei-Faktor Anmeldung (2FA)

Kimai unterstützt die Zwei-Faktor Authentifizierung mit TOTP Tokens: sichern Sie Ihre Benutzer Accounts noch heute ab!

Statistiken

Ihr Dashboard zeigt Ihnen mehrere Statistiken über aufgezeichnete Zeiten und die Menge der gesammelten Daten. Und wir können jederzeit weitere Diagramme hinzufügen!

Docker

Wir haben Docker-Images für die Entwicklung und es gibt mehrere Docker-Compose Varianten für den Produktionsbetrieb.

Stechuhr

Wollen Sie manuelle Zeiten, also willkürliche Stundenzettelaufzeichnungen, verhindern? Wechseln Sie in den Stechuhr Modus, der nur „Punch-In“- und „Punch-Out“-Aufnahmen zulässt.

Stundensatz Berechnung

Tarife können basierend auf Stunden- und Festbeträgen berechnet werden, individuell konfigurierbar für Kunden, Projekte, Aktivitäten und jeden Stundenzettel-Datensatz .

LDAP

Verbinden Sie Kimai mit Ihrem Unternehmensverzeichnisserver (LDAP oder Active Directory) und nutzen Sie Ihre vorhandene Authentifizierungsinfrastruktur.

Migration

Daten aus Kimai v1 können für eine nahtlose Migration Ihrer Zeiterfassung importiert werden.

Anzahl laufender Einträge

Sie können beliebig viele Zeiten gleichzeitig aufzeichnen, oder Sie limitieren die Anzahl paralleler Aufzeichnungen auf eine.

AM/PM Format

Kimai verwendet eine lokalisierte Version des 24-Stunden-Formats. Aber wenn Sie an das AM/PM-Format gewöhnt sind, können Sie das konfigurieren.

Nur Dauer

Wenn Ihre lokalen Gesetze verlangen, dass Sie nur die Dauer anstelle von Start- und Enddatum und Uhrzeit anzeigen, können Sie diesen Modus global aktivieren.

Suche nach Zeiten

Sie können alle Zeiterfassungen nach Benutzer, Kunde, Projekt, Aktivität und Datum filtern.

Berechtigungen

Anpassbare Berechtigungsstruktur. Weisen Sie Ihrem Benutzer verschiedene Rollen zu, z. B. können Benutzer Zeiten erfassen, Teamleiter können Rechnungen drucken und Projekte/Aktivitäten konfigurieren, Administratoren können Benutzer und Kunden konfigurieren.

Benutzerprofil

Persönliche Statistiken und Benutzereinstellungen für Stundensatz, Kimai-Design, Passwort und Avatar.

Anpassbare Darstellung

Kimai bringt verschiedene Farb- und Anzeige-Konfigurationen, sodass es jeder Benutzer an seinen persönlichen Geschmack anpassen kann

Benutzer Registrierung

Kimai unterstützt die Selbstregistrierung von Benutzern mit E-Mail-Aktivierung.

Passwort vergessen

Passwort vergessen? Kein Problem, Kimai hat eine Funktion zum Zurücksetzen des Passworts per E-Mail.
