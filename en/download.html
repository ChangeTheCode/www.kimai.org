Kimai Installation

Kimai is an open source software that is available for self-hosting as well as SaaS.

We offer several ways to install Kimai. Choose the option that suits you.

SSH

The recommended method installs Kimai directly on your server.

Documentation

Docker

Use our Docker image to install and update Kimai container easily.

Documentation

Kubernetes

You can deploy Kimai to Kubernetes using our Helm-Charts.

Kimai Cloud

No need to install Kimai, you can start using it right now.

Start for free
At this stage, it’s not possible to install Kimai by downloading a ZIP file and uploading the files to your web server. You need to have SSH access to your server. This doesn’t necessarily mean you need a VPS. Some shared hosts give you SSH access, through which you should be able to install Composer and Kimai just fine.
