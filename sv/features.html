Vad kan Kimai göra för dig?

Kimai erbjuder mycket mer än bara tidrapportering, här är en lista över andra viktiga affärsfunktioner.

Mobilklar

Layouten är helt responsiv, gjord för stationär och mobil användning. Varje användare kan konfigurera de datakolumner som ska visas, t.ex. för att förhindra horisontell rullning på små enheter.

Tillägg

Kimai byggdes för att kunna utökas med tillägg. Utvecklare som vill skapa tillägg kan använda många API-funktioner.

Fleranvändare

Kimai kan användas av så många användare som du vill: varje installation stöder obegränsade användare med olika roller (se behörigheter).

Flerspråkig

Kimai kan lokaliseras till vilket språk som helst och är redan översatt till mer än 30 språk.

Grupper

Grupper kan skapas för att organisera dina användare i logiska grupper som avdelningar eller projektrelaterade grupper. Kunder och projekt kan tilldelas dessa grupper, vilket kommer att begränsa synlighet och åtkomst för andra användare.

Kunder

Hantera dina kunder och ange adress, kontakter, land, valuta, synlighet (för arkivering) och många fler fält.

Projekt

Hantera projekt för varje kund med ordernummer, budget och synlighet (för arkivering).

Aktiviteter

Aktiviteter används för att tilldela arbetsenheten till tidrapporter. Hantera aktiviteter för varje projekt med ordernummer, budget och synlighet (för arkivering). Aktiviteter kan vara globala, du behöver inte längre kopiera om flera projekt delar samma arbetsenhet.

Fakturering

Olika fakturamallar, alternativ för postgruppering, konfigurerbara fakturanummer och du kan lägga till dina egna mallar i PDF-, DOCX-, HTML-, XLSX- och ODS-filformat.

Exportera data

Tidrapporter kan exporteras till andra program eller delas med olika avdelningar i XLSX (Excel), PDF, CSV och HTML-format. Exporterade poster kan dessutom låsas och skyddas mot ytterligare ändringar.

Utgifter

Köp tillägget och håll koll på utgifter baserat på kund, projekt och aktivitet; för kategorisering och inkludering i dina fakturor.

SAML

Logga in och registrera användare via SAML (single sign-on aka SSO) och återanvänd din befintliga identitetsleverantör, som Google Workspace, Authentik eller Azure Active Directory.

JSON API

Med ett JSON-API så att externa appar kan kommunicera med ditt favoritverktyg för tidrapportering.

Tvåfaktorsautentisering (2FA)

Kimai stödjer tvåfaktorsautentisering med TOTP-token: säkra dina användarkonton idag!

Statistik

Din överblicksbild visar statistik, registrerad tid och annat genererat data. Det finns även möjlighet att lägga till mer information som visas.

Docker

Vi har Docker-bilder för utveckling och det finns flera varianter av Docker-Compose-inställningar för produktionsanvändning.

Tid-klocka

Vill du inte att dina användare ska logga godtyckliga tidrapporter? Växla till klockläget, som endast tillåter "punch in" och "punch out" poster. Inga fler manuellt givna start- och sluttider.

Prisberäkning

Priser kan beräknas baserat på timbelopp och fasta belopp, konfigurerbara för kunder, projekt, aktiviteter och varje tidrapport individuellt.

LDAP

Anslut Kimai med din företagskatalogserver (LDAP eller Active Directory) och använd din befintliga autentiseringsinfrastruktur.

Migration

Data från Kimai v1 kan importeras, för en sömlös migrering av din tidsspårare.

Antal aktiva poster

Spela in så många gånger du vill parallellt, eller begränsa det till en.

AM/PM-format

Kimai använder en lokaliserad version av 24-timmarsformatet. Men om du är van vid AM/PM-formatet kan du konfigurera det.

Endast varaktighet

Om dina lokala lagar kräver att du bara visar varaktigheter istället för start- och slutdatum-tid, kan du aktivera det läget globalt.

Tidrapportfilter

Du kan filtrera alla tidsposter efter användare, kund, projekt, aktivitet och datum.

Behörigheter

Anpassningsbar behörighetsstruktur. Tilldela olika roller till din användare, t ex användare kan registrera tider, gruppledare kan skriva ut fakturor och konfigurera projekt/aktiviteter, administratörer kan konfigurera användare och kunder.

Användarprofil

Personlig statistik och inställningar per användare för timpris, Kimai-tema, lösenord och avatar.

Teman

Kimai kommer med olika färgteman och displayinställningar, för att passa varje enskild användare

Användarregistrering

Kimai stöder självregistrering av användare med e-postaktivering.

Lösenordsåterställning

Glömt ditt lösenord? Kimai hjälper dig att återställa den via e-post.
