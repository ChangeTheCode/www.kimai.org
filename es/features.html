Las mejores características de seguimiento de tiempo para ti

Kimai ofrece una amplia gama de características que se adaptan a tus necesidades de seguimiento de tiempo de proyectos y soluciones prácticas para nuestros clientes, lo que facilita su contabilidad.

Más características a través de complementos

Garantizo soporte para todas las aplicaciones que he construido y te asistiré a ti y a tu equipo.

¿Qué puede hacer Kimai por ti?

Kimai ofrece mucho más que solo seguimiento de tiempo, aquí tienes una lista de otras funciones empresariales importantes.

Preparado para dispositivos móviles

El diseño es completamente receptivo, hecho para uso en escritorio y dispositivos móviles. Cada usuario puede configurar las columnas de datos que se deben mostrar, por ejemplo, para evitar el desplazamiento horizontal en dispositivos pequeños.

Complementos

Kimai fue construido para ser extensible y las características se pueden agregar con complementos. Los desarrolladores que deseen crearlos pueden utilizar muchos puntos de extensión.

Usuarios múltiples

Kimai puede ser utilizado por tantos usuarios como desees: cada instalación admite usuarios ilimitados con diferentes roles (ver permisos).

Multilingüe

Kimai se puede localizar en cualquier idioma y ya está traducido a más de 30 idiomas.

Equipos

Se pueden crear equipos para agrupar a tus usuarios en grupos lógicos como departamentos o grupos de pares relacionados con proyectos. Los clientes y proyectos se pueden asignar a esos equipos, lo que limitará la visibilidad y el acceso para otros usuarios.

Clientes

Administra a tus clientes y establece dirección, contactos, país, moneda, visibilidad (para archivar) y muchos otros campos.

Proyectos

Administra proyectos para cada cliente con número de orden, presupuesto y visibilidad (para archivar).

Actividades

Las actividades se utilizan para asignar la unidad de trabajo a las entradas de la hoja de horas. Administra actividades para cada proyecto con número de orden, presupuesto y visibilidad (para archivar). Las actividades pueden ser globales, no es necesario duplicarlas si varios proyectos comparten la misma unidad de trabajo.

Facturación

Diferentes plantillas de factura, opciones para agrupación de entradas, números de factura configurables y puedes agregar tus propias plantillas en formato de archivo PDF, DOCX, HTML, XLSX y ODS.

Exportar datos

Los datos de la hoja de horas se pueden exportar a otros programas o compartir con diferentes departamentos en formato XLSX (Excel), PDF, CSV y HTML. Los registros exportados pueden estar bloqueados y protegidos contra cambios adicionales.

Gastos

Compra el complemento y lleva un registro de gastos basados en cliente, proyecto y actividad, para su categorización e inclusión en tus facturas.

SAML

Inicia sesión y registra usuarios a través de SAML (inicio de sesión único, también conocido como SSO) y reutiliza tu proveedor de identidad existente, como Google Workspace, Authentik o Azure Active Directory.

API JSON

Incluye una API JSON para que las aplicaciones externas puedan comunicarse con tu seguimiento de tiempo favorito.

Autenticación de dos factores (2FA)

Kimai admite la autenticación de dos factores con tokens TOTP: ¡protege tus cuentas de usuario hoy!

Estadísticas

Tu panel de control mostrará varias estadísticas sobre los tiempos registrados y la cantidad de datos recopilados. ¡Y siempre podemos agregar más gráficos!

Docker

Tenemos imágenes de Docker para desarrollo y múltiples variaciones de configuraciones de Docker-Compose para uso en producción.

Reloj de tiempo

¿No quieres que tus usuarios registren registros arbitrarios de hoja de tiempo? Cambia al modo de reloj de tiempo, que solo permite registros de 'entrada' y 'salida'. No más horarios de inicio y finalización dados manualmente.

Cálculo de tarifas

Las tarifas se pueden calcular en función de montos por hora y fijos, configurables para clientes, proyectos, actividades y cada registro de hoja de tiempo de manera individual.

LDAP

Conecta Kimai con el servidor de directorio de tu empresa (LDAP o Active Directory) y utiliza tu infraestructura de autenticación existente.

Migración

Los datos de Kimai v1 se pueden importar para una migración sin problemas de tu seguimiento de tiempo.

Cantidad de registros activos

Puedes registrar tantas veces como desees en paralelo, pero también puedes configurar el límite permitido de registros activos hasta 1.

Formato AM/PM

Kimai utiliza una versión localizada del formato de 24 horas. Pero si estás acostumbrado al formato AM/PM, puedes configurarlo.

Solo duración

Si las leyes locales requieren que muestres solo duraciones en lugar de fecha y hora de inicio y finalización, puedes activar ese modo globalmente.

Filtro de hoja de tiempo

Puedes filtrar todos los registros de tiempo por usuario, cliente, proyecto, actividad y fecha.

Permisos

Estructura de permisos personalizable. Asigna roles diferentes a tus usuarios, por ejemplo, los usuarios pueden registrar tiempos, los líderes de equipo pueden imprimir facturas y configurar proyectos/actividades, los administradores pueden configurar usuarios y clientes.

Perfil de usuario

Estadísticas personales y configuraciones por usuario para tarifa por hora, tema de Kimai, contraseña y avatar.

Temas

Kimai cuenta con diferentes temas de color y configuraciones de visualización para adaptarse a cada usuario individual

Registro de usuario

Kimai admite el auto-registro de usuarios con activación por correo electrónico.

Restablecimiento de contraseña

¿Olvidaste tu contraseña? Kimai te ayuda a restablecerla por correo electrónico.
