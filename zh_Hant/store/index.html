附加元件、應用程式、支援

透過我們精選的外掛程式來強化 Kimai。精選自 Kimai 生態系統中的數十種應用程式與擴充功能，立即讓您擁有更專業的工時追蹤體驗！

超過 2.2K+ 名使用者信任 Kimai
Featured

  • 工作時數、休假、病假、法定假日

    員工管理：工作與加班時數。假期、病假與請假。法定假日管理與行事曆整合。月份審核與 PDF 輸出。 繼續閱讀

  • 自訂欄位

    建立可自由設定的額外欄位，適用於工時單、客戶、專案、活動與使用者，並支援各種格式。欄位可設為選填或必填，且可設定存取權限。 繼續閱讀

  • 開銷管理

    根據客戶與專案追蹤您的開銷，可分類並列入收支單。 繼續閱讀

  • Kiosk 模式 —— 支援掃描條碼、RFID 標籤與 Web NFC 標籤

    在使用者不便進入 Kimai 的環境（例如倉庫）中，透過使用條碼或 RFID 掃描器來開始與結束工時表紀錄。 繼續閱讀

  • 審查軌跡紀錄

    記錄工時單、客戶、項目及活動的詳細變更/審計日誌，並逐項顯示在時間軸上。 繼續閱讀

  • 任務規劃

    建立任務，開始、停止、暫停任務，分派給使用者與團隊並記錄其工時 —— 使用 Kimai 管理待辦事項清單 繼續閱讀

  • 自訂翻譯

    使用您的翻譯自訂您的 Kimai 安裝實例中的所有可見元素，包含標籤、表格標題列、選單項目、等…… 繼續閱讀

  • 支援自動產生 PDF 工時單的收支單

    強化收支單的各功能。切換貨幣與日期的語言地區設定，對新的收支單自動產生 PDF 工時證明單。 繼續閱讀

  • 自訂內容

    建立自訂的 CSS 樣式表、JavaScript 手槁、全域警告訊息與能以 Markdown 格式撰寫公司最新消息的新頁面。 繼續閱讀
