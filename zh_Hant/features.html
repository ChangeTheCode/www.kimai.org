Kimai 能為您做些什麼？

Kimai 不僅僅提供工時追蹤，還有其它重要的業務功能。

行動裝置友善

版面配置完全採用回應式設計，同時適用於桌上型電腦與行動裝置。每位使用者可以設定要顯示的資料縱行，可避免在小型裝置上需要橫向捲動等情況。

外掛程式

Kimai 具有可擴充的設計，可透過外掛程式增加功能。想要建立外掛程式的開發人員能夠利用多個擴充點。

多使用者

Kimai 允許任意數量的使用者：每個已安裝實例都支援無上限的使用者與各種團隊角色（請參閱「權限」）。

多語言

Kimai 可以本地化為任何語言，目前已獲翻譯為至少 30 種語言。

團隊

可以建立團隊以將您的使用者合理分組，例如依部門或與專案相關的同儕小組分組。可將客戶與專案指派給這些團隊，以限制對其他使用者的可見度與存取權。

客戶

管理您的客戶並設定地址、聯絡人、國家、貨幣、可見度（用於封存）與更多其它欄位。

專案

透過訂單編號、預算與可見度（封存用）來管理每位客戶的專案。

活動

活動用於將工作單元指派到工時單中的項目。透過訂單編號、預算與可見度（封存用）來管理每項專案的活動。活動可以設定為全域，因此多項專案共用同一個工作單元時無須複製活動。

收支單

各種收支單模版、項目分組選項、可設定的收支單編號，並可新增自己的 PDF、DOCX、HTML、XLSX 與 ODS 檔案格式的模版。

匯出資料

工時單資料可以匯出給其它程式處理，或使用 XLSX (Excel)、PDF、CSV 與 HTML 檔案格式與不同部門共用。還可以鎖定已匯出的紀錄以防止進一步更改。

開銷

購買該外掛程式後，可以根據客戶、項目與活動來追蹤開銷 ，以便分類並列入您的收支單。

SAML

利用 SAML 式單一登入（SSO），透過您現有的身份識別提供者，例如 Google Workspace、Authentik 或 Microsoft Entra ID，來登入與註冊。

JSON API

具有能用以讀取資料的全面 JSON API，因此外部應用程式與您的自訂工具能與您所喜好的工時追蹤工具溝通。

雙重要素驗證（2FA）

Kimai 支援使用 TOTP token 進行雙重要素身份認證（2FA）：立即保護您的使用者帳號！

統計資訊

您的儀錶板會顯示有關已記錄工時與收集資料量等的統計資訊。而我們能持續新增更多種圖表的支援！

Docker

我們有用於開發的 Docker 映像，還有為正式用途所設定的多種 Docker-Compose 變體。

Time-clock

Don’t want your users to log arbitrary timesheet records? Switch to the time-clock mode, which only allows “punch in” and “punch out” records. No more manually given start and end times.

Rate calculation

Rates can be calculated based on hourly and fixed amounts, configurable for customers, projects, activities and each timesheet record individually.

LDAP

將 Kimai 與您的公司目錄伺服器（LDAP 或 Active Directory）連接並使用您現有的身份驗證基礎架構。

版本遷移

支援匯入 Kimai v1 的資料，以便從您的工時追蹤工具無縫遷移。

活動紀錄數

可允許平行記錄任意次，也可限制為一次。

AM/PM format

Kimai uses localized version of the 24 hour format. But if you are used to the AM/PM format, you can configure that.

僅列出持續時長

如果您當地的法律要求您只列出持續時長而不是列出開始與結束時間，您可以全域啟用本模式。

Timesheet filter

You can filter all time-records by user, customer, project, activity and date.

Permissions

Customizable permission structure. Assign different roles to your user, e.g. users can record times, teamleads can print invoices and configure project/activities, admins can configure users and customers.

User profile

Personal statistics and per user settings for hourly rate, Kimai theme, password and avatar.

Theming

Kimai comes with different color themes and display settings, to suit each individual user

User registration

Kimai supports user self-registration with e-mail activation.

Password reset

Forgot your password? Kimai helps you reset it via e-mail.
