Open Source

Kimai 是套開放源始碼的工時追蹤解決方案。Kimai 能夠追蹤工作時間並根據您的需求列印出您的活動摘要，可按年度、月度、日度、客戶、專案、等……。簡單就是 Kimai 的優點。由於 Kimai 的介面利用網頁瀏覽器的排版系統打造，Kimai 支援多種平臺，行動裝置也不例外。

Kimai 不僅減少了將打卡鐘卡片上的工時資料複製到 Excel 試算表的乏味過程，更提供了其它許多您不知道的有趣功能！

為您的商業成功出一份力

Kimai 能提供您設定工時追蹤流程所需的所有核心業務功能，並可與您現有的基礎設施整合。

認證與安全

Kimai 支援外部身份識別提供者，可從多個提供者（如 Google Workspace、Azure AD 或 Authentik）透過 LDAP 與 SAML 進行登入。此外還可透過 TOTP token 啟用雙重要素身份認證。

報告與分析

分析已記錄的工時、客戶、專案、活動。可根據使用者、客戶、專案、活動、標籤、時間區間、等條件進行評級。

收支單

各種收支單模版、項目分組選項、可設定的收支單編號，並可新增自己的 PDF、DOCX、HTML、XLSX 與 ODS 檔案格式的模版。

JSON API

具有能用以讀取資料的全面 JSON API，因此外部應用程式與您的自訂工具能與您所喜好的工時追蹤工具溝通。

Kimai 能為您做些什麼？

Kimai 不僅僅提供工時追蹤，還有其它重要的業務功能。

外掛程式

Kimai 具有可擴充的設計，可透過外掛程式增加功能。想要建立外掛程式的開發人員能夠利用多個擴充點。

多使用者

Kimai 允許任意數量的使用者：每個已安裝實例都支援無上限的使用者與各種團隊角色（請參閱「權限」）。

多語言

Kimai 可以本地化為任何語言，目前已獲翻譯為至少 30 種語言。

團隊

可以建立團隊以將您的使用者合理分組，例如依部門或與專案相關的同儕小組分組。可將客戶與專案指派給這些團隊，以限制對其他使用者的可見度與存取權。

客戶

管理您的客戶並設定地址、聯絡人、國家、貨幣、可見度（用於封存）與更多其它欄位。

專案

透過訂單編號、預算與可見度（封存用）來管理每位客戶的專案。

收支單

各種收支單模版、項目分組選項、可設定的收支單編號，並可新增自己的 PDF、DOCX、HTML、XLSX 與 ODS 檔案格式的模版。

匯出資料

工時單資料可以匯出給其它程式處理，或使用 XLSX (Excel)、PDF、CSV 與 HTML 檔案格式與不同部門共用。還可以鎖定已匯出的紀錄以防止進一步更改。

開銷

購買該外掛程式後，可以根據客戶、項目與活動來追蹤開銷 ，以便分類並列入您的收支單。

SAML

利用 SAML 式單一登入（SSO），透過您現有的身份識別提供者，例如 Google Workspace、Authentik 或 Microsoft Entra ID，來登入與註冊。

JSON API

具有能用以讀取資料的全面 JSON API，因此外部應用程式與您的自訂工具能與您所喜好的工時追蹤工具溝通。

雙重要素驗證（2FA）

Kimai 支援使用 TOTP token 進行雙重要素身份認證（2FA）：立即保護您的使用者帳號！

統計資訊

您的儀錶板會顯示有關已記錄工時與收集資料量等的統計資訊。而我們能持續新增更多種圖表的支援！

Docker

我們有用於開發的 Docker 映像，還有為正式用途所設定的多種 Docker-Compose 變體。

Time-clock

Don’t want your users to log arbitrary timesheet records? Switch to the time-clock mode, which only allows “punch in” and “punch out” records. No more manually given start and end times.
