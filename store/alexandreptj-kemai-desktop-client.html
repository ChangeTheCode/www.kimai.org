Kemai — Desktop client

Cross-platform GUI client for Kimai

Offered by
Alexandre Petitjean
free
Download
  • Description

Kemai is a cross-platform (Windows, OSX, Linux) desktop client for Kimai.

Features

Kemai aims to ease regular task on Kimai :

  • Start / Stop activity
  • Create customer, project and activity
  • Start / Close task from Task Management plugin
