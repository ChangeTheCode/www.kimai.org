O que o Kimai pode fazer por você?

O Kimai oferece muito mais do que apenas o controle de horas. Aqui está uma lista de outras funções comerciais importantes.

Pronto para dispositivos móveis

O layout é totalmente responsivo, feito para desktop e uso móvel. Cada usuário pode configurar as colunas dos dados que devem ser mostrados, para evitar a rolagem horizontal em dispositivos pequenos por exemplo.

Plug-ins

O Kimai foi desenvolvido para ser extensível e os recursos podem ser adicionados através de plug-ins. Os desenvolvedores que desejam criá-los podem usar muitos pontos de extensão.

Multiusuário

O Kimai pode ser usado por quantos usuários você quiser: cada instalação não possui um limite de usuários e com diferentes funções (consulte permissões).

Vários idiomas

O Kimai pode ser localizado em qualquer idioma e já está traduzido para mais de 30 línguas.

Equipes

As equipes podem ser criadas para agrupar os seus usuários em grupos lógicos, como departamentos ou grupos de pares relacionados ao projeto. Os clientes e os projetos podem ser atribuídos a estas equipes, o que limitará a visibilidade e o acesso dos outros usuários.

Clientes

Gerencie os seus clientes e defina um endereço, um país, uma moeda, visibilidade (para arquivamento), contatos e muitos outros campos.

Projetos

Gerencie os projetos para cada cliente com número de pedido, orçamento e visibilidade (para arquivamento).

Atividades

As atividades são usadas para atribuir a unidade de trabalho às entradas do quadro de horários. Gerencie as atividades de cada projeto com o número do pedido, orçamento e visibilidade (para arquivamento). As atividades podem ser globais, sem necessidade de duplicação caso vários projetos compartilhem a mesma unidade de trabalho.

Faturamento

Diferentes modelos de fatura, opções para agrupamento das entradas, fatura com números configuráveis e você pode adicionar os seus próprios modelos em formato PDF, DOCX, HTML, XLSX e ODS.

Exportar dados

Os dados do quadro de horários podem ser exportados para outros programas ou podem ser compartilhados entre diferentes departamentos nos formatos XLSX (Excel), PDF, CSV e HTML. Os registros exportados também podem ser bloqueados e protegidos contra outras alterações.

Despesas

Compre o plug-in e acompanhe as despesas com base no cliente, no projeto e na atividade - para a categorização e para a inclusão nas suas faturas.

SAML

Faça login e registre os usuários via SAML (single sign on, também conhecido como SSO) e reutilize o seu provedor de identidade já existente, como o Google workspace, Authentik ou o active directory Azure.

API JSON

Apresenta uma extensa API JSON para leitura e gravação de dados, de modo que aplicativos externos e suas ferramentas personalizadas possam se comunicar com o seu rastreador de tempo favorito.

Autenticação de dois fatores (2FA)

O Kimai oferece suporte à autenticação de dois fatores com tokens TOTP: proteja as suas contas de usuário hoje mesmo!

Estatísticas

O seu painel mostrará várias estatísticas sobre os tempos registrados e a quantidade dos dados coletados. E nós sempre poderemos adicionar mais gráficos!

Docker

Temos imagens do Docker para desenvolvimento e diferentes variações de configurações do Docker-Compose para uso em produção.

Relógio de ponto

Não deseja que os seus usuários registrem um quadro de horários arbitrários? Mude para o modo relógio-de-ponto, isso permite apenas os registros “de entrada” e “de saída”. Chega de horários de início e término fornecidos manualmente.

Cálculo da taxa

As taxas podem ser calculadas com base em valores horários e valores fixos, configuráveis para clientes, para projetos, para atividades e para cada registro individual no quadro de horários.

LDAP

Conecte o Kimai com o servidor da sua empresa (LDAP ou Active Directory) e use a sua própria infra-estrutura de autenticação.

Migração

Os dados do Kimai v1 podem ser importados, para uma migração perfeita do seu rastreador de tempo.

Quantidade de registros ativos

Você pode fazer registros em paralelo quantas vezes quiser, ou, limite-os para apenas um.

Formato AM/PM

O Kimai usa uma versão localizada do formato de 24 horas. Mas se você está acostumado com o formato AM/PM, também poderá configurá-lo.

Apenas duração

Se as leis locais exigirem que você mostre apenas as durações em vez da data e hora de início e término, você pode ativar esse modo globalmente.

Filtro do quadro de horários

Você pode filtrar todos os registros de tempo por usuário, por cliente, por projeto, por atividade e pela data.

Permissões

Estrutura de permissões personalizáveis. Atribua diferentes funções ao seu usuário, por exemplo, os usuários podem registrar tempos, os líderes de equipe podem imprimir faturas e configurar os projetos/atividades, os administradores podem configurar os usuários e os clientes.

Perfil do usuário

Estatísticas pessoais e configurações individuais para a taxa horária, para o tema do Kimai, para a senha e o avatar.

Temas

O Kimai vem com diferentes temas de cores e configurações de exibição, para se adequar a cada usuário individualmente

Registro de usuário

O Kimai suporta o auto-registro do usuário com ativação por e-mail.

Redefinir a senha

Esqueceu a sua senha? O Kimai ajuda você a redefini-la via e-mail.
