Instalação do Kimai

O Kimai é um programa de código aberto que está disponível para hospedagem própria e SaaS.

Oferecemos várias maneiras de instalar o Kimai. Escolha a opção que mais lhe convém.

SSH

O método recomendado é instalar o Kimai diretamente no seu servidor.

Documentação

Docker

Use nossa imagem Docker para instalar e atualizar facilmente o contêiner do Kimai.

Documentação

Kubernetes

Você pode implantar o Kimai no Kubernetes usando nossos Helm Charts.

Documentação

Kimai Cloud

Não é necessário instalar o Kimai, você pode começar a usá-lo agora mesmo.

Comece gratuitamente
Nesta fase, não é possível instalar o Kimai baixando um arquivo ZIP e enviando os arquivos para o seu servidor web. Isso ocorre porque o Composer precisa ser executado via linha de comando. Isso não significa necessariamente que você precisa de um VPS. Alguns hosts compartilhados fornecem acesso SSH, por meio do qual você poderá instalar o Composer e o Kimai sem maiores problemas.
